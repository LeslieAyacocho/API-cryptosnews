# CryptoNews

CryptoNews is a system output for System Architecture and Integration Course (TUP-T 2021)

Technologies used:
- Laravel 8
- API's (CoinRanking APi & CryptoCompare News API)
- Bootstrap
- Google Chart API
- MySQL
- jQuery
